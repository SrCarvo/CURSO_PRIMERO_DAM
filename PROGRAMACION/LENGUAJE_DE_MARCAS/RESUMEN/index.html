INTRODUCCIÓN A HTML

HTML (HyperText Makeup Languaje) es un lenguaje de marcado usado en las paginas web con la principal ventaja que este permite enlazar documentos mediante hipervínculos y permite incluir contenido multimedia


Para empezar, HTML presenta una estrucutra constante. Empezariamos creado una etiqueta "<html> </html>", la cual quedaria dividida en "<head> </head>" y "<body> </body>". (La etiquetas empiezan con "< >" y acaban con < />)


La declaracion en HTML5 se define como <!DOCTYPE html> El codigo base, quedaría algo como así
<!DOCTYPE html>
<html>
<head>
<title>Ejemplo de HTML</title> <meta charset="UTF-8">
</head>
<body>
<h1>Hola, mundo</h1> <p>Bienvenido a mi página web.</p>
</body> </html>
